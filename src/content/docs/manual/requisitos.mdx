Para utilizar el compilador de LibreScript, su sistema debe cumplir con los siguientes requisitos:

## Hardware

- **Procesador:** Cualquier procesador moderno (Intel Core i3 o equivalente AMD, o superior).
- **RAM:** 4 GB de RAM 4 GB de RAM como mínimo (8 GB o más recomendados para un rendimiento óptimo).
- **Almacenamiento:** 500 MB de espacio libre en disco.

## Software

- **Sistema Operativo:**
  - Windows 10 o posterior.
  - macOS 10.15 (Catalina) o posterior.
  - Distribuciones de Linux modernas (Ubuntu 20.04+, Fedora 34+, Debian 11+, etc.).
- **Node.js:** Versión 14.x o superior. Node.js es necesario para ejecutar el compilador, ya que está desarrollado en JavaScript.
  - Puede descargarlo desde el sitio web oficial de Node.js: https://nodejs.org/
  ![Alt text](/AstroLibresScript/img/requisitos/nodejs.webp)
- **npm (Node Package Manager):** 
  - Generalmente viene incluido con Node.js. Se utiliza para gestionar las dependencias del proyecto.
  ![Alt text](/AstroLibresScript/img/requisitos/npm.webp)
- **Editor de Código, se recomienda utilizar el editor de código Visual Studio Code.**
  - Este editor es una herramienta ligera pero potente que se ejecuta en el escritorio y es compatible con los sistemas operativos Windows, macOS y Linux. Puedes descargarlo desde el siguiente enlace oficial: https://code.visualstudio.com/download
  ![Alt text](/AstroLibresScript/img/requisitos/vsc.jpg)

## Instalación
Siga estos pasos para instalar el compilador de LibreScript

<Steps>
1. **Clonar o Descargar el Repositorio:** Obtenga los archivos del compilador. Si está utilizando Git, puede clonar el repositorio: [Repositorio LibreScript](https://github.com/Ness024/LibreScript-IDE)

    ---

    ![Repositorio de librescript codigo](/AstroLibresScript/img/requisitos/clonarrepo1.jpg)
    
    ---
    
    ![Repositorio de librescript ZIP](/AstroLibresScript/img/requisitos/clonarrepo2.png)
    
    ---

    ABRIMOS EL EXPLORADOR DE ARCHIVOS Y CREAMOS UNA CARPETA EN EL APARTADO DE NUESTRA CONVENENCIA:

    ![Explorador de archivos](/AstroLibresScript/img/requisitos/exploradorarchivos.png)
    
    ---

    ![Nueva carpeta](/AstroLibresScript/img/requisitos/carpetanueva.png)
    
    ---

    ``` bash
    git clone https://github.com/Ness024/LibreScript-IDE
    cd <directorio_del_repositorio>
    ```

    ---

    ![terminal](/AstroLibresScript/img/requisitos/terminal.jpg)
    
    ---

    :::note
    Si descargó un archivo ZIP, descomprímalo en la ubicación deseada y navegue hasta el directorio del proyecto.
    ![librescript en zip](/AstroLibresScript/requisitos/img/zip.png)
    :::

2. **Instalar Dependencias:** Una vez dentro del directorio del proyecto (donde se encuentra `package.json`). 
    ![Package.json](/AstroLibresScript/img/requisitos/packagejson.png)
    
    Instale las dependencias necesarias utilizando `npm install` dentro de la terminal:

    ```bash
    npm install
    ```

    ---

    ![Instalando dependencias](/AstroLibresScript/img/requisitos/npminstall.png)

    ---

    Este comando instalará `moo` (para el lexer) y `nearley` (para el parser), tal como se especifica en `package.json`.

3. **Compilar la Gramática:** El parser de LibreScript utiliza `nearley`. Debe compilar la gramática de Nearley antes de poder ejecutar el compilador. Esto generará el archivo `grammar.js` a partir de `grammar.ne` (o `grammar.txt` si lo renombró).
    
    ```bash
    npm run build:gramar 
    ```

    ---

    ![npm run](/AstroLibresScript/img/requisitos/npmrun.png)

    ---

    Este comando ejecuta nearleyc `grammar.ne` -o `grammar.js`. Asegúrese de que su archivo de gramática se llame `grammar.ne` o ajuste el comando en `package.json` si tiene un nombre diferente (por ejemplo, `grammar.txt`).

    ---

    ![Compilando gramática](/AstroLibresScript/img/requisitos/runnearley.png)

    ---
    
    ¡Con estos pasos, el compilador estará listo para ser utilizado!

4. **Live Server**
    Buscamos la extensión de LIVE SERVER en Visual Studio Code.
    ![Live Server](/AstroLibresScript/img/requisitos/extliveserver.png)
    
    ---

    Instalamos la extensión y abrimos el proyecto en Visual Studio Code.

    ![Instalando Live Server](/AstroLibresScript/img/requisitos/installliveserver.png)

5. **Ejecutamos nuestro `LibreScript` nos vamos a la carpeta de `LibreSriptIDE.html`**
    
    ![LiveServer](/AstroLibresScript/img/requisitos/opnwls.png)

    ---

6. **!ESTA LISTO PARA ESCRIBIR CODIGO¡**
    ![LiveServer](/AstroLibresScript/img/requisitos/ide.png)

</Steps>
