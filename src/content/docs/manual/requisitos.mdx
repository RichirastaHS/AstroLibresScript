## Hardware

- Procesador: Cualquier procesador moderno (Intel Core i3 o equivalente AMD, o superior).
- RAM: 4 GB de RAM como mínimo.
- Almacenamiento: 10 MB de espacio libre en disco.

## Software

- Sistema Operativo:
  - Windows 10 o posterior.
  - macOS 10.15 (Catalina) o posterior.
  - Distribuciones de Linux modernas (Ubuntu 20.04+, Fedora 34+, Debian 11+, etc.).
  -Editor de Texto o IDE:** Cualquier editor de código como:
    - Visual Studio Code (recomendado)
    - Sublime Text
    - Atom
  - Node.js: Versión 14.x o superior. Node.js es necesario para ejecutar el compilador, ya que está desarrollado en JavaScript.
  - npm (Node Package Manager): Generalmente viene incluido con Node.js. Se utiliza para gestionar las dependencias del proyecto.

## Instalación

Siga estos pasos para instalar el compilador de LibreScript

<Steps>
1. **Clonar o Descargar el Repositorio:** Obtenga los archivos del compilador. Si está utilizando Git, puede clonar el repositorio:

    ``` bash
    git clone [https://github.com/AdrCrz/librescript.git](https://github.com/AdrCrz/librescript.git)
        cd librescript
    ```
    :::note
    Si descargó un archivo ZIP, descomprímalo en la ubicación deseada y navegue hasta el directorio del proyecto.
    :::

2. **Instalar Dependencias:** Una vez dentro del directorio del proyecto (donde se encuentra package.json), instale las dependencias necesarias utilizando npm:

    ```bash
    npm install
    ```
    Este comando instalará moo (para el lexer) y nearley (para el parser), tal como se especifica en package.json.

3. **Verificar la instalación:** Para asegurarte de que todo está configurado correctamente, puedes intentar analizar el archivo de ejemplo `test.ls`.
    ```bash
    node main.js
    ```
    ¡Con estos pasos, el compilador estará listo para ser utilizado!
</Steps>
